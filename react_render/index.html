ReactDOM.render()

A função ReactDOM.render() renderiza HTML para a página da Web. A função recebe dois argumentos, sendo o primeiro código HTML e o segundo o local do código HTML onde o React será redenrizado

Exemplo: 

    <script type="text/babel">
        var local = document.getElementById("exemplos")

        ReactDOM.render(
        <p>Texto redenrizado</p>, 
        local)
    </script>
