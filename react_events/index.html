React Events

O React pode executar ações com base nos eventos do usuário. Os eventos React são escritos na sintaxe camelCase: onClick em vez de onclick. Os manipuladores de eventos são escritos entre chaves: onClick={shoot} em vez de onClick="shoot()". Para passar um argumento para um manipulador de eventos, usa-se uma função de seta. São três abordagens diferentes:

  • No código HTML - Adicionado ao nó precedido da palavra ‘on’ e chamando a função COM parênteses - Ex.: <button onclick=“funcao()”>botão</button>
  • No DOM Tradicional - Adicionado ao nó precedido da palavra ‘on’ ex: onclick e chamando a função SEM parênteses (elemento.evento = nomeDaFunção) - Ex.: document.getElementById("botao").onclick = funcao
  • Ouvintes de eventos DOM - Adiciona um ouvinte de eventos ao nó de elemento DOM (elemento.addEventListener(‘evento’, nomeDaFunção, booleano)), sem o "on" e chamando a função SEM parênteses - Ex.: document.getElementById("botao").addEventListener('click', funcao, false). ou:

         }

            render(){

                    <>

                    </>

            }
        }
