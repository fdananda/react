React Components

Os componentes são bits de código independentes e reutilizáveis. Os componentes são em dois tipos, componentes de classe e componentes de função.

  • Componente de classe - Deve incluir a instrução "extends React.Component". Esta instrução cria uma herança para React.Component e requer um método render(), retornando HTML.
  • Componente de Função - Também retorna HTML, porém sem necessidade de um render(), e se comporta da mesma maneira que um componente Class, mas podem ser escritos usando muito menos código.

Exemplos: 

    <script type="text/babel">
        var local1 = document.getElementById("exemplo1")
        var local2 = document.getElementById("exemplo2")

        //Class Component
        class Pessoa1 extends React.Component{
        	render(){
        		return <p>Nome: Fernanda</p>
        	}
        }

        //Function Component
        function Pessoa2(){
        		return <p>Nome: Helena</p>
        }

        ReactDOM.render(<Pessoa1/>, local1)
        ReactDOM.render(<Pessoa2/>, local2)
    </script>
