React CSS Styling

Existem muitas maneiras de estilizar o React com CSS, dentre elas: Estilo embutido / folhas de estilo CSS / Módulos CSS. Em relação ao CSS convencional, as propriedades com separadores de hífen (ex.: background-color) devem ser escritas com a sintaxe camel case (backgroundColor).

Exemplos: 

    <script type="text/babel">
       var local = document.getElementById("exemplo")

        //Class Component
        class Pessoa extends React.Component{

            render(){

             var exemploStyle = {
               color: "black",
               fontFamily: "Arial",
               backgroundColor: this.props.bgcolor,
               padding: 5
            }

                return <p style={exemploStyle}>Nome: {this.props.children} 

            }
        }

        ReactDOM.render(
        <div>
            <Pessoa bgcolor="#58B3FF">Fernanda</Pessoa>
            <Pessoa bgcolor="#FF605F">Helena</Pessoa>
            <Pessoa bgcolor="#FFD52E">Rodrigues</Pessoa>
            <Pessoa bgcolor="#AE99FF">Moreira</Pessoa>
        </div>
        , local)
    </script>
